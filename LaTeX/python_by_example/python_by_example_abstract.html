Zusammenfassung

Die Sprache Python wurde Anfang der 1990er Jahre von Guido van Rossum in Amsterdam entwickelt. Mitlerweile steht hinter der Sprache die Python Software Foundation mit der eigenen Open-Source Python License. Der Name referiert die britische Komikertruppe Monty Python, nicht etwa die gleichnamige Schlangengattung der Pythons. In der Dokumentation sind daher viele Anspielungen auf Sketche aus dem Flying Circus.
Python wurde mit dem Ziel entworfen, möglichst einfach und übersichtlich zu sein. Dies wird durch zwei Dinge erreicht. Zum einen enthält die Sprache einen sehr mächtigen Funktionsumfang in nur wenigen Schlüsselwörtern, zum anderen wurde die Syntax reduziert und auf Übersichtlichkeit optimiert. Dies führt dazu, dass Python eine Sprache ist, in der man schnell, einfach und leicht programmieren kann.

Ziel dieses Vortrags ist eine Einführung in die Sprache anhand einiger Quellcode-Beispiele. Diese sollen verdeutlichen, wie einfach und elegant die Sprache durch ihren Minimalismus und ihrem funktionalen Anteil ist.

Abstract

The language Python was developed in the early 1990s by Guido van Rossum in Amsterdam. Now behind the language is the Python Software Foundation with its own open source Python License. The name speaks the british troupe Monty Python comedian, not the same species of snake-Pythons. The documentation contains many allusions to sketches from the Flying Circus.
Python was created with the aim designed simply and clearly as possible to be. This is achieved by two things. First, the language contains very powerful features in a few keywords, on the other hand, the syntax was reduced and clarity improved. The result is that Python is a language in which one quick, simple and easy to program.

The aim of this presentation is an introduction into the languages of some source code examples. These will show how simple and elegant python is by its minimalism and its functional part.
